SV-Customs Base Registries [Motivation] ()
NARRATIVE: The SV-Customs Base Registries [Motivation] view extends Semantic view and provides a comprehensive framework for leveraging data objects and business objects related to customs activities. This view is divided into three main elements: the Customs Base Registry, the Customs Information Base, and the Base Registry. The Customs Base Registry is a Grouping ABB that encompasses six Data Object SBBs, each representing a different system or database used in customs activities. These include the REX - Registered Exporter system, EORI - Economic Operators Registration and Identification number, ICS (Import Control System), TARIC (Integrated Tariff of the European Union), VAT Information Exchange System (VIES), and AEO - Authorised Economic Operator. Each of these systems or databases serves a unique purpose in the customs process, from certifying the origin of goods and assigning unique identification numbers to economic operators, to enhancing customs control, providing tariff information, verifying VAT numbers, and recognizing businesses that meet certain criteria and demonstrate compliance with customs regulations and security standards. The Customs Information Base is a Diagram Model-group ABB that includes nine Business Object ABBs, each representing a different type of information used in customs activities. These include Exports information, Imports information, Origin of goods Information, Economic value Information, Customs partners Information, Binding Tariffs Information, Economic Operators Registration and Identification number (EORI), Economic Operators Information, and Workforce information. These business objects provide essential information for customs activities, such as details about exports and imports, the origin of goods, economic value, customs partners, binding tariffs, registration and identification numbers for economic operators, information about economic operators, and workforce information. The Base Registry is a Data Object ABB that refers to reliable sources of basic information that permits an efficient, secure, and transparent exchange of the data. This registry serves as a foundational element in the SV-Customs Base Registries [Motivation] view, providing the underlying data that supports the other elements in the view. In summary, the SV-Customs Base Registries [Motivation] view provides a comprehensive framework for managing and leveraging data and information related to customs activities. It integrates various systems and databases, each serving a unique purpose in the customs process, and provides a wide range of essential information for customs activities. The Base Registry serves as the foundation for this view, providing reliable sources of basic information that enable efficient, secure, and transparent data exchange.
