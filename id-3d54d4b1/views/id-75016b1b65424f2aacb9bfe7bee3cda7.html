TVA-Customs Data Exchange Enablers [Motivation] ()
NARRATIVE: The TVA-Customs Data Exchange Enablers [Motivation] view extends Technical view - application and provides a comprehensive framework for leveraging digital tools and components that facilitate the exchange of data between systems or applications. This view is primarily composed of two main elements: 'Data Exchange Enablers' and 'Customs Data Exchange Solutions'. The 'Data Exchange Enablers' is a Grouping ABB that refers to tools or components that facilitate the exchange of data between systems or applications. Within this grouping, there are two key elements: 'Data Exchange' as an Application Component ABB and 'Data Exchange' as an Application Service ABB. The 'Data Exchange' Application Component ABB enables the implementation of the procedure to transfer data between different individuals, organizations, or systems. On the other hand, the 'Data Exchange' Application Service ABB enables the secure exchange of messages, records, forms, and other kinds of data between different individuals, organizations, or systems. This includes data routing, except endpoint discovery. The second main element is the 'Customs Data Exchange Solutions', another Grouping ABB. This grouping contains a key element named 'Central system for the standardized exchange of information', which is an Application Service SBB. The main objective of this system is to make all INF data available and to streamline the processes of INF data management. For this purpose, the European Commission has designed an EU centralized INF SP System. Additionally, there is an 'Once-Only Technical System', an Application Service ABB, which enables the sharing of information between public administrations across borders between EU countries. It is cross-sectorial and can be expanded beyond the current scope of life events set out in the Single Digital Gateway Regulation. It puts into practice the Once-Only Principle, which states that citizens should not be forced to provide information to authorities if another authority already holds that information in electronic format. In summary, the TVA-Customs Data Exchange Enablers [Motivation] view provides a detailed framework for data exchange enablers and customs data exchange solutions, highlighting the importance of efficient and secure data exchange between different entities and systems.
