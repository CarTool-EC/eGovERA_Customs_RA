dct:titleCouncil Regulation (EEC) No 2658/87 of 23 July 1987 on the tariff and statistical nomenclature and on the Common Customs Tariff
dct:descriptionRegulation (EEC) No 2658/87 is a European Union regulation that establishes the goods nomenclature called the 'Combined Nomenclature', or 'CN', which meets at one and the same time, the requirements both of the Common Customs Tariff and of the EU's external trade statistics. It is used to classify goods for customs and statistical purposes within the European Union.

skos:exampleFor instance, if a company wants to import a certain type of machinery into the EU, it would use the Combined Nomenclature established by Regulation (EEC) No 2658/87 to determine the correct classification code for that machinery. This code would then be used to calculate the customs duties that need to be paid and to record the trade for statistical purposes.
eira:concepteira:SolutionBuildingBlock
skos:noteThe regulation is designed to facilitate the uniform application of customs tariffs and statistical nomenclatures within the EU. It provides a systematic framework for the classification of goods, which is essential for customs duties, statistical analyses, and trade negotiations. The regulation is updated annually to reflect changes in technology, trade patterns, and the needs of users.
