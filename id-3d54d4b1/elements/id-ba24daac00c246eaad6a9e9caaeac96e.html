Customs Decisions system ()
Definition: The Customs Decisions System (CDS) is an IT concept that refers to a digital system used by customs authorities to manage and process customs decisions. It is designed to streamline and simplify the process of making customs decisions, making it more efficient and effective. Source: European Commission Source reference: https://ec.europa.eu/taxation_customs/business/customs-procedures/general-overview/customs-decisions-system_en Additional information: The Customs Decisions System is a part of the Union Customs Code (UCC) which aims to modernize customs procedures within the European Union. The system provides a centralized platform for customs authorities and economic operators to submit, process, and manage customs decisions and simplifications. It also ensures transparency and legal certainty by providing a clear overview of all decisions taken. The system covers all types of customs decisions including binding tariff information, binding origin information, and economic operator registration and identification. Example: 1. An importer can use the Customs Decisions System to apply for a binding tariff information decision, which provides certainty about the customs duties that will be paid on a specific product. 2. A customs authority can use the system to process and manage applications for customs simplifications, such as simplified declaration procedures or self-assessment. 3. An exporter can use the system to check the status of their application for a binding origin information decision.
