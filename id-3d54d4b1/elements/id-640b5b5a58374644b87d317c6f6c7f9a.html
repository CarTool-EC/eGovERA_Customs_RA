Customs Reference Data ()
Definition: Customs Reference Data is a grouping that covers a comprehensive set of elements related to customs procedures and regulations. It includes information on customs offices, product classification codes, country codes, currency codes, units of measure, legal references, and customs valuation methods. These elements are essential for ensuring compliance with customs requirements and procedures, facilitating international trade, and enabling unambiguous electronic communication of quantities together with their units. Source: EIRA Team Research Additional information: The Customs Reference Data catalogue includes elements such as the Reference Data Customs Offices List, which provides information on the Reference Data (RD) and the Customs Office List (COL) used by national customs applications. It also includes the Harmonized System (HS) Codes, an internationally standardized system for classifying products and goods. The ISO Country Codes, which are internationally recognized codes that designate every country and most of the dependent areas, are also part of this catalogue. The catalogue also includes the Currency Code List, the Unified Code for Units of Measure (UCUM), Legal and Regulatory References, and Customs Valuation Methods. Example: For instance, the Harmonized System (HS) Codes are used to determine the customs duties, taxes, and regulations applicable to specific products during import or export. Similarly, the ISO Country Codes are used in top-level domains on the Internet and at international events. The Currency Code List is used to list concepts associated with currencies and currency subunits, while the Unified Code for Units of Measure (UCUM) is used to facilitate unambiguous electronic communication of quantities together with their units.
